    \title{High Power LED Controller}
    \author{Joe Colosimo // jwcxz.com}
    \maketitle

    \begin{abstract}\centering
        This LED controller, which is part of the ACRIS
        Project,\footnote{http://jwcxz.com/projects/acris} provides the
        capability to control high power LEDs by sending serial data over a
        simple RS-485 network.  Special emphasis was placed on making the board
        easy to install and communicate with.  It is intended to relieve
        hobbyists and designers from the burden of developing boards to control
        their LEDs so that they may focus on more creative aspects of design.
        Full schematics and code for the project are available at
            http://jwcxz.com/projects/acris/ledctrlr.php.
    \end{abstract}
